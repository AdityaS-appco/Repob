# Aditya Singh - Professional Portfolio

This is a high-performance, technical masterpiece portfolio built using pure HTML, CSS, and vanilla JavaScript.

## Architecture
- **index.html**: Semantic structure and content.
- **style.css**: Modern styling with CSS Grid, Flexbox, Custom Properties, and Glassmorphism.
- **script.js**: Interactive features including a custom cursor, scroll progress, and reveal animations.

## Key Features
- High-end dark aesthetic.
- Custom interactive cursor (Desktop).
- Scroll-based reveal animations using Intersection Observer.
- Responsive design for all devices.
- Integrated professional data for Aditya Singh.
